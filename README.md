# Knowledge Base Agent 

Fully Local RAG Knowledge Base Agent for Business Operations category. Answers HR policy questions from company documents (PDFs/TXTs).

## Features
- Semantic Search on company documents
- Precise Answers (1-2 relevant lines only)
- 100% FREE & LOCAL (no API keys/cloud)
- Gradio Web UI (professional demo)
- Processes PDFs + TXT files
